# 🚀 Automação E2E com Playwright

Projeto de automação de testes End-to-End desenvolvido com **Playwright** utilizando **JavaScript**, com foco em boas práticas.

## 📌 Objetivo

Criar uma suíte de testes automatizados cobrindo fluxos críticos de uma aplicação web de e-commerce, simulando cenários reais do dia a dia.

---

## 🧪 Cenários automatizados

### 🔐 Login
- Login com sucesso
- Login com credenciais inválidas

### 🛒 Carrinho
- Adicionar produto ao carrinho
- Remover produto do carrinho

### 💳 Checkout
- Finalizar compra com sucesso
- Validação de erro ao não preencher dados obrigatórios

---

## 🛠️ Tecnologias utilizadas

- JavaScript
- Node.js
- Playwright
- GitHub Actions (CI/CD)
